# Campus Rush

Campus Rush 是一款使用 Phaser 3 和 Arcade Physics 制作的 2D 横版校园无尽跑酷游戏。玩家扮演即将迟到的学生，通过跳跃和下蹲躲避不同类型的校园障碍，尽可能坚持更久并刷新本机最高分。

## 环境要求

- Node.js 22 或更高版本
- npm
- 普通桌面版 Chrome 或 Edge

## 本地运行

```bash
npm install
npm run dev
```

打开开发服务器输出的本地地址即可游玩。

## 操作

- 空格键：跳跃
- 方向上键：跳跃
- 点击或轻触游戏画面：跳跃
- 按住方向下键或 S：在地面下蹲
- M 键：开启或关闭音效
- 获得弹力运动鞋后，可在空中额外跳跃一次

## 构建与预览

```bash
npm run build
npm run preview
```

生产文件输出到 `dist`。项目使用相对资源路径，可部署到 GitHub Pages 或其他静态托管服务。

## GitHub Pages 部署

项目已经包含 `.github/workflows/deploy.yml`。将代码推送到 GitHub 仓库的 `main` 分支后：

1. 在仓库的 **Settings → Pages** 中将 Source 设为 **GitHub Actions**。
2. 推送代码，或在 Actions 页面手动运行部署工作流。
3. 等待 `Deploy Campus Rush to GitHub Pages` 完成，再打开工作流提供的网址。

工作流使用 `npm ci`、`npm run build` 和 `dist` 静态产物，不需要后端或环境变量。

## 游戏规则

- 角色自动向前奔跑，场景和障碍向左滚动。
- 书包、路障、水坑、交通锥和课本等地面障碍需要跳跃。
- 低垂横幅和低垂树枝只能下蹲通过；跳跃会撞上障碍。
- 飞来的篮球既可以跳过，也可以下蹲躲开。
- 开始、跳跃、下蹲、成功越障、分数里程碑和游戏结束均有程序化音效；音效开关会保存在本机。
- 金币每枚增加 10 分，磁力学生卡可自动吸取附近金币。
- 校徽护盾可抵挡下一次碰撞，双倍学分会临时翻倍所有得分。
- 早八咖啡会让角色进入高速无敌冲刺并撞飞所有障碍。
- 弹力运动鞋会临时解锁二段跳。
- 所有技能都有时间限制，重复拾取同种道具会刷新持续时间。
- 分数按生存时间增加，也可通过金币和撞飞障碍获得额外分数。
- 游戏速度逐步提升，并在可操作范围内封顶。
- 碰到任何障碍后本局结束，可立即重新开始。

## 2–3 分钟展示顺序

1. 展示开始页、操作说明和本机最高分。
2. 开始游戏，依次演示键盘和点击跳跃。
3. 演示地面障碍与低垂障碍的不同躲避方式、实时分数和速度提升。
4. 主动碰撞，展示本局分数与最高分。
5. 连续重新开始，说明旧障碍和计时状态不会残留。

## 范围说明

当前版本不包含永久道具升级、商店、音效、音乐、账号、排行榜、联机、多角色或多地图。

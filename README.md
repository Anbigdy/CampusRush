# Campus Rush

Campus Rush 是一款使用 Phaser 3 和 Arcade Physics 制作的 2D 横版剧情跑酷游戏。玩家扮演即将迟到的学生，通过跳跃和下蹲躲避障碍；达到 1000 分后会触发命运事件，穿越到银月异世界继续奔跑。

## 环境要求

- Node.js 22 或更高版本
- npm
- 普通桌面版 Chrome 或 Edge

## 本地运行

```bash
npm install
npm run dev
```

打开开发服务器输出的本地地址即可游玩。

## 操作

- 空格键、方向上键或点击画面：跳跃；在空中再次操作可二段跳
- 按住方向下键或 S：在地面下蹲；在空中会立即快速下坠
- M 键：开启或关闭背景音乐与音效
- Esc 键：暂停游戏；解除暂停后经过 3 秒倒计时再继续

## 构建与预览

```bash
npm run build
npm run preview
```

生产文件输出到 `dist`。项目使用相对资源路径，可部署到 GitHub Pages 或其他静态托管服务。

## GitHub Pages 部署

项目已经包含 `.github/workflows/deploy.yml`。将代码推送到 GitHub 仓库的 `main` 分支后：

1. 在仓库的 **Settings → Pages** 中将 Source 设为 **GitHub Actions**。
2. 推送代码，或在 Actions 页面手动运行部署工作流。
3. 等待 `Deploy Campus Rush to GitHub Pages` 完成，再打开工作流提供的网址。

工作流使用 `npm ci`、`npm run build` 和 `dist` 静态产物，不需要后端或环境变量。

## 游戏规则

- 角色自动向前奔跑，场景和障碍向左滚动。
- 触屏移动设备进入游戏后会在画面下方显示虚拟按键：点按右键跳跃/二段跳，按住左键下蹲或在空中快速下坠；桌面端不会显示。
- 书包、路障、水坑、交通锥和课本等地面障碍需要跳跃。
- 叠高的器材箱和高书车超过单次跳跃高度，必须使用二段跳。
- 低垂横幅和低垂树枝只能下蹲通过；跳跃会撞上障碍。
- 飞来的篮球既可以跳过，也可以下蹲躲开。
- 120 分后会程序生成斜坡、连续山丘、跳台缺口、平台障碍和空中速降等上层路线，不再只轮换固定模板。
- 路线按当前速度、章节和玩家近期表现调整难度，并在进入、爬升、恢复、高潮、出口等分段中安排挑战；300 分后可能出现持续 8–14 秒的长高台路线。
- 每条程序路线会在当前速度、提速后和最高速度下验证跳跃可达性及最小操作窗口；验证失败会自动重生成，最终仍失败则回退到安全模板。
- 每组高台路线都保证放置金币；金币只铺在平台障碍前后的安全区间，高层平台需要主动跳跃或二段跳才能获取完整奖励。
- 特殊道具大礼包会在高难度峰顶出现，并带有保底机制：连续三组路线未出现时，下一组必定生成礼包路线。
- 大礼包固定提供一次性永久护盾、至少 50 分奖励和另外两种随机临时技能。
- 默认背景音乐为 ConcernedApe 的 `Summer (Nature's Crescendo)`；首次点击开始后播放并循环。
- 开始、跳跃、下蹲、收集金币、获得技能、成功越障、分数里程碑和游戏结束均有程序化音效；声音开关会保存在本机。
- Snow Peak 的中文台词显示在气泡中，并使用浏览器本机的通用中文语音清晰朗读；若存在中文男声会优先选用，同时用低音量 CC0 小猫录音作“哈基米”点缀。浏览器不支持语音合成时回退为猫叫，不包含真人录音或克隆音色。
- 金币每枚增加 10 分，磁力学生卡可自动吸取附近金币。
- 金币按角色的真实跳跃抛物线排列，并会为前后障碍预留安全间距。
- 校徽护盾拾取后不会随时间消失，可永久待命并抵挡下一次碰撞；双倍学分会临时翻倍所有得分。
- 早八咖啡会让角色进入高速无敌冲刺并撞飞所有障碍。
- 校园盲盒使用独立于普通技能的生成计时器，每 7–10 秒尝试出现一次；有 2/3 概率开出随机像素哈基米彩蛋，权重是其他结果合计的 2 倍，也可能获得奖学金、临时技能、校园卡欠费或“谢谢参与”。哈基米揭晓按柔和 30%、活泼 30%、爆炸 40% 分层随机：柔和档不闪屏、不震动，活泼档适度跳转，爆炸档保留漩涡、横穿、陨石砸落、镜像翻牌等全屏冲击。
- 校园卡欠费只会让接下来 5 枚金币失效，不会改变角色物理状态或导致无解死亡。
- 临时技能都有时间限制，重复拾取同种道具会刷新持续时间。
- 校园篇达到 1000 分后会触发一次性主线：卡车撞击、传送门过场与异世界场景切换。
- 进入异世界后保留本局分数和速度，背景与障碍会替换为月夜王国、史莱姆、符文石碑、魔晶尖塔和影蝠。
- 分数按生存时间增加，也可通过金币和撞飞障碍获得额外分数。
- 游戏速度逐步提升，并在可操作范围内封顶。
- 若物理碰撞异常导致角色进入斜坡、高台或地面内部，系统会解除错误支撑并立即恢复到正常地面状态，避免卡进地下后无敌。
- 碰到任何障碍后本局结束，可立即重新开始。

程序路线可通过 `npm run validate:routes` 批量检查多档速度下的可达性、礼包保底、金币数量和金币/障碍安全间距。

Snow Peak 候选台词中的“选择比努力更重要”和“方向比努力更重要”取自张雪峰署名书名（[图书信息](https://book.douban.com/subject/35548274/)）；“放平心态，戒骄戒躁”“做好自己该做好的事情”“充满信心就对了”可在其[本人微博主页](https://weibo.com/u/1676679984)核对。其余短句是游戏已有的网络梗化文案，不作为逐字原话引用。

## 2–3 分钟展示顺序

1. 展示开始页、操作说明和本机最高分。
2. 开始游戏，依次演示键盘和点击跳跃。
3. 演示斜坡、必须跳跃的多层平台、快速下坠路线和峰顶特殊道具大礼包。
4. 演示地面障碍与低垂障碍的不同躲避方式、实时分数和速度提升。
5. 达到 1000 分，展示卡车撞击、传送门与异世界切换。
6. 在异世界继续跑酷，再主动碰撞展示本局分数与最高分。
7. 重新开始，说明新一局会从校园篇起点开始且旧状态不会残留。

## 范围说明

当前版本不包含永久道具升级、商店、账号、排行榜、联机、多角色或更多可选地图。
